# Prajit Narule Portfolio

A clean, responsive one-page portfolio website for Prajit Narule, a short-form video editor. The site presents Prajit's positioning, five embedded YouTube Shorts samples, differentiators, services, and direct contact links.

## Key Technologies

- TanStack Start with React 19
- TanStack Router file-based routing
- Vite
- Tailwind CSS 4
- Lucide React icons
- Netlify deployment configuration

## Local Development

Install dependencies, then start the development server:

```bash
npm install
npm run dev
```

The Netlify build pipeline validates production builds automatically.
